# 更新日志

## 0.1.0 - 2026-07-07

### 新增

- 识别中文法条引用。
- 支持条、款、项级别解析。
- 在编辑模式和阅读模式中 hover 预览法条正文。
- 提供右侧 `Legal References` 面板。
- 支持按引用 occurrence 显示 `已插入 / 未插入`。
- 支持 callout、nested bullet、block quote、plain text 四种插入格式。
- 支持仅插入法条正文，省略插件生成的法律名称和条文序号题头。
- 支持 `Ctrl+Enter` 自动补全法条正文。
- 支持从 Word、PDF、TXT、Markdown 导入法条来源文件。
- 内置七部法学生常用法律的 starter library。
- 支持法律别名配置，例如 `刑诉法=刑事诉讼法`。

### 说明

- 这是面向中文法学生笔记工作流的早期版本。
- 内置法条库用于学习便利；正式引用请核对官方文本。
